# HybrIQ Confluence Starter

AI agents for Confluence powered by [HybrIQ](https://hybriq.dev) (Atlassian Forge macros).

- **Spec Review** — Analyze documents for gaps, ambiguities, and missing acceptance criteria
- **Knowledge Q&A** — Answer questions from Confluence page content with source citations
- **Space Digest** — Weekly digest of space activity, new pages, and discussions

## Quick Start

```bash
git clone https://github.com/developerlabsai/hybriq-confluence-starter.git
cd hybriq-confluence-starter && cp .env.example .env
npm install && forge register && forge deploy && forge install
```
